## 种植技巧

- 每天浇水可缩短生长时间
- 使用肥料提高作物品质
- 温室可以无视季节限制
- 作物成熟后及时收获，否则会枯萎

::: warning 注意
每个季节最后一天记得收获所有作物，换季时未收获的作物会枯萎！
:::
