# 赚钱攻略

## 稳定收入来源

| 方式 | 日收益 | 难度 |
|------|--------|------|
| 种植作物 | 500-2000G | 简单 |
| 钓鱼卖鱼 | 300-1500G | 中等 |
| 完成任务 | 200-1000G | 简单 |
| 手工艺制作 | 800-3000G | 中等 |

## 进阶赚钱技巧

1. **季节限定作物** — 种植当季高价值作物，如秋季南瓜
2. **稀有鱼类** — 雨天钓稀有鱼，售价翻倍
3. **批量制作** — 原材料加工后售价更高
4. **活动参与** — 节日活动奖励丰厚

::: tip 最佳赚钱路线
早晨收菜 → 制作加工品 → 钓鱼到中午 → 卖鱼 → 下午做任务 → 晚上参加活动
:::
